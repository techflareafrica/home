# TechFlare Africa Website

A complete static website with a responsive corporate homepage and blog.

## Files
- `index.html` — main website
- `blog/index.html` — blog listing
- `blog/posts/cloud-computing-african-businesses.html` — published article
- `blog/posts/cloud-computing-african-businesses.md` — Markdown source
- `assets/css/style.css` — all styling
- `assets/js/main.js` — mobile menu, copyright year, and email contact form

## Publish on GitHub Pages
1. Upload all files to the root of the `techflareafrica/home` repository.
2. In GitHub, open **Settings → Pages**.
3. Choose **Deploy from a branch**, then select `main` and `/ (root)`.
4. The site will be available at `https://techflareafrica.github.io/home/`.

## Before launch
- Replace `hello@techflareafrica.com` if a different email is required.
- Review service wording and contact details.
- The contact form opens the visitor's email application; connect a form service or backend if direct submission is required.

## Added features
- Dark mode toggle with system preference detection and saved visitor choice.
- Newsletter subscription component on every page. It currently demonstrates validation and stores sign-ups locally in the browser; connect it to a real email marketing provider before launch.
- Footer links for LinkedIn, X, Facebook, and GitHub. Replace the general platform URLs with TechFlare Africa's official profile URLs.
